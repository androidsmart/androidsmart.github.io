## How To Flash
Flash ROMs GApps steps:
1. Flash preflash
2. Flash custom rom
3. Format data
4. Reflash Recovery
5. Reboot System

Flash ROMs Vanilla steps:
1. Flash preflash before flash rom
2. Flash ROM Vanilla build
3. Reflash Recovery
4. Reboot Recovery
5. Format data
5. Flash RO2RW DFE
6. Reboot Recovery again
7. Flash Gapps (Recomended Using nikGapps)
8. Wipe data, metadata, dalvik, cache
9.Reboot System
