## Install
### Flashing steps
- fastboot flash boot boot.img
- fastboot reboot recovery
- on recovery go to Format Data > Factory Reset
- go to Apply Update > Apply from ADB 
- adb sideload rom.zip
- go to Apply Update > Apply from ADB 
- adb sideload gapps.zip
- reboot
- enjoy! :D
